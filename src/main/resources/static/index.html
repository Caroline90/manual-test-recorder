Load the unpacked extension from this repository, open the popup, click Start picker, and then click any element on the page you want to document. Each picked element is sent to /api/events and appears below as an XRAY-ready step.

  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Choose Load unpacked and select this repository folder.
  4. Use the popup to start the picker and capture elements from the current tab.
